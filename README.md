# mucoki

A modern blog website built with HTML, CSS, and JavaScript.

## Overview

mucoki is a clean, responsive blogging platform designed for sharing thoughts and ideas.

## Features

- ✨ Responsive design that works on all devices
- 🎨 Modern and clean UI
- 📖 Easy-to-read typography
- 🌙 Dark mode support
- ⚡ Fast loading times
- 🔍 SEO optimized

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Deploy to GitHub Pages or any web server

## File Structure

```
mucoki/
├── index.html              # Main homepage
├── blog.html               # Blog posts list
├── css/
│   ├── style.css           # Main styles
│   └── responsive.css      # Mobile responsive styles
├── js/
│   └── script.js           # Main functionality
├── package.json            # Project metadata
├── .gitignore              # Git ignore rules
└── README.md               # This file
```

## Deployment

This site is deployed via GitHub Pages. Any push to the `main` branch will automatically deploy to `https://mucokiblogs.github.io/The-Mucoki/`
