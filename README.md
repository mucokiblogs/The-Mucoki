# mucoki - Modern Blogging Platform

A modern, SEO-optimized blog website built with HTML, CSS, and JavaScript.

## Overview

mucoki is a clean, responsive blogging platform designed for sharing thoughts and ideas with excellent search engine optimization (SEO) for Google rankings.

## Features

- ✨ Responsive design that works on all devices
- 🎨 Modern and clean UI with optimal UX
- 📖 Easy-to-read typography and content structure
- 🌙 Dark mode support with theme persistence
- ⚡ Fast loading times and optimized performance
- 🔍 **SEO Optimized** for Google rankings
- 📱 Mobile-first responsive design
- 🎯 Structured data and schema.org markup
- 🔗 Semantic HTML5 for better crawlability

## SEO Features Implemented

### On-Page SEO
- Meta tags (title, description, keywords)
- Open Graph tags for social sharing
- Twitter Card tags
- Canonical URLs
- Schema.org structured data (JSON-LD)
- Semantic HTML5 markup
- Proper heading hierarchy (H1, H2, H3)
- Alt text for images
- Internal linking strategy

### Technical SEO
- `robots.txt` for crawler instructions
- `sitemap.xml` for search engine discovery
- Fast page load times
- Mobile-responsive design
- Clean URL structure
- Proper redirects

### Content SEO
- Keyword-rich content
- Proper heading tags
- Internal linking
- Blog posts with meta descriptions
- Fresh, unique content

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Deploy to Cloudflare or your hosting platform
4. Submit sitemap to Google Search Console

## Deployment on Cloudflare

1. Connect your GitHub repository to Cloudflare Pages
2. Set build command: `npm run build` (optional)
3. Set publish directory: `/`
4. Deploy the site

## Google Search Console Setup

1. Verify your site ownership in Google Search Console
2. Submit your sitemap: `https://mucoki.com/sitemap.xml`
3. Monitor search performance and rankings
4. Check for any crawl errors or warnings

## Core Web Vitals Optimization

- **LCP (Largest Contentful Paint)**: < 2.5s
- **FID (First Input Delay)**: < 100ms
- **CLS (Cumulative Layout Shift)**: < 0.1
